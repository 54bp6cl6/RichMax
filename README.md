# RichMax
大富翁用電子錢包 Line Bot 是一款可以增加大富翁遊戲樂趣的 Line bot，  
讓玩家們可以用數位化的方式管理遊戲資產，
讓遊戲過程更輕鬆也更好玩。

主要功能有：
- 領錢、繳錢、匯錢等交易功能
- 交易撤銷與交易歷史紀錄
- 刺激有趣的機會命運卡牌
